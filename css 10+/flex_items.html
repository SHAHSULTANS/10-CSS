order

By default, items are displayed in the order they appear in the source code, but the 'order' property allows you to change this without altering the HTML structure.

One
Two
Three

Application

  • An e-commerce platform,online stores reorder products to highlight specific items like top sales or customer favorites.

HTML Snapshot for order


<div class="flex-container">
    <div style="order: 2;">One</div>
    <div style="order: 3;">Two</div>
    <div style="order: 1;">Three</div>
</div>

flex-grow

The flex-grow property in CSS Flexbox defines how much a flex item will grow relative to the rest of the flex items inside a flex container.

One
Two
Three

Application

  • Dashboard Widgets
  • Responsive Footers etc

HTML Snapshot for flex-grow


<div class="flex-container">
    <div style="flex-grow:0.3">One</div>
    <div style="flex-grow:0.7">Two</div>
    <div style="flex-grow:8">Three</div>
</div>
