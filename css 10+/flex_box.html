CSS Flexbox is a powerful layout system that offers a flexible and efficient way to arrange elements within a container. It easier to design a flexible responsive layout structure without using float or positioning.

Common Use Cases

  • Creating navigation menus
  • Building responsive Grid
  • Aligning elements within containers
  • Designing card layout

Example

Here are three flex items(white color). Flex items are declared within a flex container(aqua color).

One
Two
Three

HTML


<div class="flex-container">
    <div>One</div>
    <div>Two</div>
    <div>Three</div>
</div>

CSS


.flex-container{
    display: flex;
    background-color: aqua;
    padding: 10px;
}
.flex-container div{
    width: 100px;
    line-height: 100px;
    background-color: white;
    margin:10px;
    text-align: center;
}

Application

Dynamic Content Generation

Loops help generate HTML content dynamically, like creating table rows or list items based on data fetched from an API.

    
const users = ['Alice', 'Bob', 'Charlie'];
let userList = '';
for (let user of users) {
    userList += `<li>${user}</li>`;
}
document.getElementById('user-list').innerHTML = userList;

Form Validation

    
const fields = document.querySelectorAll('input');
for (let field of fields) {
    if (!field.value) {
        alert(`${field.name} is required`);
    }
}

API Data Handling

After fetching data from an API, loops are used to proccess and display this data

    
fetch('https://api.example.com/users')
.then(response => response.json())
.then(users => {
    for (let user of users) {
        console.log(user.name);
    }
});
